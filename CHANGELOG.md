# Changelog

All notable changes to the Ulrich Energy Auditing website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive CI/CD pipeline with GitHub Actions
- Automated Lighthouse CI performance monitoring
- Security scanning with npm audit and gitleaks
- Jest and Playwright testing framework
- Docker Compose configuration
- Enhanced nginx configuration with security headers
- Automated backup and rollback scripts
- Monitoring setup with 5-minute health checks
- Complete documentation suite

## [1.0.0] - 2025-03-11

### Added
- Initial website launch
- 4-page static site (Home, About, Services, Contact)
- Responsive design with Tailwind CSS
- WCAG 2.1 AA compliant color palette
- Next.js 16 with App Router
- Static export deployment
- Docker/nginx hosting on Unraid server
- Builder-focused messaging throughout
- 45L tax credit information
- Minnesota Green Path partnership details
- Contact information and CTAs

### Technical
- Next.js 16.1.6 with Turbopack
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui components
- Lucide React icons
- Geist font family

### Deployment
- Docker containerization
- nginx static file serving
- Unraid server hosting (192.168.1.203:8088)
- Automated deployment scripts
- Health check monitoring
